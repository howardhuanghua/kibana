[[timelion]]
= 时序控件

[partintro]
--
时序控件（Timelion）是一个基于时间序列的数据可视化工具，它可以将多种独立的数据源合并呈现到一张视图上。用户可以利用一些简单的表达式语言提取所需数据，并进行一些计算将复杂的问题简化并呈现结果。

例如，Timelion 可以非常轻松的解决这些问题：

* 过去某段时间页面的 UV 量是多少？
* 本周五和上周五的流量有多少差异？
* 本站今天来自日本的访客占多少百分比？
* 标普500指数过去10天的移动平均值是多少？
* 过去两年所有的查询请求总量有多少？
